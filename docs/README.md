# Documentation

This folder holds repo documentation that used to live at the project root.

- `specs/` - method and format specifications.
- `validation/` - validation memos and claim audits.
- `model-cards/` - draft model-card material for Hugging Face or related use.
- `notes/` - loose research notes that are useful to keep but are not canonical
  specs.

Use `validation/` and `results/` together before updating public claims.
